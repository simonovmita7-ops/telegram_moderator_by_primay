# Telegram Moderator Bot

Автономный Telegram-бот модератор групп с ИИ-анализом сообщений, автоматическими наказаниями и управлением через личные сообщения.

## Возможности

- ИИ-модерация (OpenAI или Claude) с контекстом последних 20 сообщений
- Автоматические предупреждения, муты, кики и баны по правилам
- Локальный антиспам-детектор
- Режим ЧС (экстренная модерация)
- Управление через ЛС: группы, настройки, статистика, логи
- Планировщик с восстановлением задач после перезапуска
- SQLite + SQLAlchemy (async)

## Требования

- Python 3.12+
- Токен Telegram-бота ([@BotFather](https://t.me/BotFather))
- API-ключ OpenAI или Anthropic Claude

## Установка

```bash
# 1. Перейти в каталог проекта
cd telegram-moderator-bot

# 2. Создать виртуальное окружение
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Настроить переменные окружения
copy .env.example .env   # Windows
# cp .env.example .env   # Linux/macOS

# 5. Отредактировать .env — вставить токены
```

## Настройка `.env`

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от BotFather |
| `AI_PROVIDER` | `openai` или `claude` |
| `OPENAI_API_KEY` | Ключ OpenAI (если openai) |
| `CLAUDE_API_KEY` | Ключ Anthropic (если claude) |
| `DATABASE_URL` | URL SQLite (по умолчанию `sqlite+aiosqlite:///bot/data/moderator.db`) |
| `LOG_LEVEL` | `DEBUG`, `INFO`, `WARNING`, `ERROR` |
| `LOG_FILE` | Путь к файлу логов |

## Запуск

```bash
python -m bot.main
```

## Подключение к группе

1. Добавьте бота в группу
2. Назначьте **администратором** с правами:
   - Удаление сообщений
   - Блокировка участников
   - Закрепление сообщений
   - Приглашение пользователей (для invite-ссылок после кика)
3. Откройте бота в ЛС → `/start` → **Мои группы**

## Управление (только ЛС)

После `/start`:

- **Мои группы** — список групп для управления
- **Настройки** — глобальная справка
- **Статистика** — сводка по всем группам
- **Логи** — последние события
- **Помощь** — инструкция

### Панель группы

- Настройки модерации (муты, кики, ИИ, спам, слова, правила)
- Нарушители и история
- Режим ЧС
- Активные муты / кики / баны
- ИИ-модерация
- Администраторы

## Команды

| Команда | Где | Описание |
|---|---|---|
| `/start` | ЛС | Главное меню |
| `/addword слово` | ЛС | Добавить запрещённое слово во все ваши группы |

## Правила наказаний

| Нарушение | 1-е | 2-е | 3-е |
|---|---|---|---|
| Оскорбление семьи | Кик 1 день | Кик 3 дня | — |
| Спам | Предупреждение | Мут 3ч | Кик 3ч |
| 18+ контент | Предупреждение | Кик 2 дня | — |
| Медиа без согласия | Предупреждение | Кик 1 день | — |
| Конфликт | Предупреждение | Мут 1 день | — |
| Слив переписки | Кик 3 дня | — | — |
| Опрос-нарушение | Мут 1 день | Кик 2 дня | — |
| Угрозы / насилие | **Мгновенный бан** | — | — |

## База данных

Таблицы создаются автоматически при первом запуске. Полная SQL-схема — в `schema.sql`.

Таблицы: `users`, `groups`, `group_admins`, `violations`, `warnings`, `mutes`, `kicks`, `bans`, `settings`, `logs`, `ai_decisions`, `chat_message_cache`.

## Планировщик

После перезапуска бот автоматически:

- Снимает истёкшие муты
- Разбанивает после киков и отправляет invite-ссылку в ЛС
- Снимает закреплённые предупреждения
- Снимает временные баны

## Логи

Логи пишутся в файл (`LOG_FILE`) и в таблицу `logs`. Режим логирования настраивается в панели группы: `full`, `minimal`, `off`.

## Режим ЧС

Экстренный режим (только владелец):

- Усиленные наказания
- Мгновенный бан при повторных нарушениях
- Агрессивный антиспам (5 сообщений вместо 15)
