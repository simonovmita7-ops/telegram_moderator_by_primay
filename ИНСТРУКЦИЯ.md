## ℹ️ Общая информация

**Group Controls Bot** — это многофункциональный Telegram-бот для автоматической модерации чатов и групп. 

Бот работает на базе 100% бесплатных мощных ИИ-движков (**Groq Cloud Llama-3** и **Google Gemini**), автоматически выдает предупреждения, муты или кики, а также предоставляет интерактивный веб-интерфейс (Mini App) для визуального управления.

---

## 🤖 Бесплатная ИИ-Модерация

Администратор каждой группы может выбрать наиболее подходящую бесплатную нейросеть прямо во вкладке **🧠 Мозг** в Mini App:

1. 🚀 **Groq Llama-3 (`llama-3.3-70b-versatile`):**
   - *Сверхбыстрые LPU-чипы:* мгновенный отклик 0.1 сек и обработка до 800 токенов в секунду.
   - *100% Бесплатно:* до 30 запросов в минуту / 14 400 в день.
2. 🧠 **Google Gemini (`gemini-2.5-flash`):**
   - *Умный нейронный движок:* глубокая аналитика по тексту правил от Google.

> **Надёжность:** Если выбранная нейросеть временно недоступна или превысила лимиты, бот автоматически переключится на фоллбек-провайдера Gemini, исключая пропуски нарушений!

## ⚡ Быстрый запуск

### 1. Подготовка окружения
```bash
git clone https://github.com/your-repo/telegram-moderator-bot.git
cd telegram-moderator-bot

python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/macOS:
source venv/bin/activate

pip install -r requirements.txt
```

### 2. Получение ключей (оба бесплатные)
1. **Telegram Bot Token:** Получите у [@BotFather](https://t.me/BotFather) в Telegram.
2. **Google Gemini API Key:** Получите в [Google AI Studio](https://aistudio.google.com/).
3. **Groq Cloud API Key:** Получите в [Groq Console](https://console.groq.com/).

## 📱 Управление через Mini App

В боте вызовите меню или нажмите кнопку **📱 Mini App**.

### Вкладки Mini App:
1. **📊 Дашборд (Главная):**
   - Сводка статистики (Нарушения, Предупреждения, Муты, Кики).
   - Список последних инцидентов с разворачиваемой панелью.
   - Кнопка **«➕ Добавить бота»** для быстрого подключения к новым группам.
2. **📋 Правила:**
   - Редактирование индивидуального текста правил для выбранной группы.
   - Переключатели категорий модерации ИИ.
3. **🧠 Мозг (ИИ):**
   - **Выбор провайдера ИИ:** переключение между **Groq Llama-3** и **Google Gemini**.
   - Плавный тумблер включения/выключения ИИ-модератора.
   - Просмотр ошибок запросов к ИИ и кнопка отправки логов автору (`@PR1MAY`).
4. **🚫 Слова:**
   - Управление списком немедленно запрещённых слов и белым списком исключений.
5. **🏆 Лидерборд:**
   - Рейтинг лучших дуэлянтов группы.
