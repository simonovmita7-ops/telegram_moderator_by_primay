# 🚀 Инструкция по развертыванию Mini App на Vercel

Эта инструкция объясняет, как запустить веб-интерфейс (Mini App) на бесплатном хостинге Vercel без использования ngrok.

---

## 📋 Требования

1. **Vercel Аккаунт** (создается бесплатно на [vercel.com](https://vercel.com)).
2. **Публичный адрес бота** (хостинг бота).
   - Бот должен быть запущен на сервере (например, VPS/VDS) с публичным IP-адресом или доменным именем.
   - Порт `8765` должен быть открыт для входящих соединений.
   - Бот может работать и на локальном компьютере, но в таком случае для внешнего доступа к API Mini App вам все равно потребуется ngrok или Cloudflare Tunnel.

---

## 🛠 Вариант 1: Прямое подключение через CORS (Рекомендуется)

Этот способ самый простой. Поскольку мы добавили CORS-заголовки в aiohttp-сервер бота, Mini App на Vercel может отправлять запросы напрямую на ваш VPS.

### Шаг 1. Укажите адрес сервера в коде Mini App
1. Откройте файл `bot/web/static/index.html`.
2. Найдите в самом начале тега `<script>` (строка ~515) переменную `API_BASE`:
   ```javascript
   const API_BASE = ""; 
   ```
3. Впишите туда публичный адрес вашего запущенного бота (IP и порт `8765` или домен):
   ```javascript
   const API_BASE = "http://ВАШ_IP_АДРЕС_VPS:8765";
   ```
   *Пример:* `const API_BASE = "http://95.213.255.12:8765";`

### Шаг 2. Загрузка на Vercel
1. Установите [Vercel CLI](https://vercel.com/cli) (через npm: `npm install -g vercel`) или используйте интеграцию с GitHub.
2. В терминале в корне проекта выполните команду:
   ```bash
   vercel
   ```
3. Авторизуйтесь, подтвердите создание нового проекта (по умолчанию для всех вопросов нажимайте Enter).
4. После окончания сборки Vercel выдаст вам публичную ссылку на Mini App.
   *Пример:* `https://telegram-moderator-by-primay.vercel.app/`

---

## 🛠 Вариант 2: Перенаправление запросов через `vercel.json` (Proxying)

Если вы не хотите менять исходный код `index.html` или хотите скрыть реальный IP-адрес вашего сервера в коде, можно настроить редирект на уровне Vercel.

### Шаг 1. Настройте файл `vercel.json`
1. Откройте файл `vercel.json` в корне проекта.
2. Найдите секцию `routes`:
   ```json
   {
     "src": "/api/(.*)",
     "dest": "https://your-bot-backend.com/api/$1"
   }
   ```
3. Замените `https://your-bot-backend.com` на реальный адрес вашего VPS (IP:порт или домен):
   ```json
   {
     "src": "/api/(.*)",
     "dest": "http://ВАШ_IP_АДРЕС_VPS:8765/api/$1"
   }
   ```
4. Сохраните изменения.

### Шаг 2. Развертывание
1. Запустите команду `vercel` в терминале в корне проекта.
2. Готово! Запросы к `/api/...` будут автоматически пересылаться Vercel-сервером на ваш VPS.

## 📱 Подключение Mini App к Telegram-боту

После того как вы получили рабочую ссылку от Vercel (например, `https://telegram-moderator-by-primay.vercel.app/`):

1. Перейдите в Telegram к боту **@BotFather**.
2. Введите команду `/setmenubutton` и выберите вашего бота.
3. Отправьте ссылку, полученную от Vercel.
4. Введите название кнопки меню (например, `🛡️ Панель` или `Group Controls`).
5. Теперь в левом нижнем углу чата с ботом появится кнопка для мгновенного запуска Mini App!

## ⚠️ Безопасность и HTTPS
Telegram требует, чтобы все ссылки Mini App работали по безопасному протоколу **HTTPS**. 
Vercel предоставляет бесплатный SSL-сертификат для вашей ссылки Mini App автоматически. 
Однако, если ваш VPS-сервер работает по HTTP (`http://ip:8765`), браузеры могут блокировать запросы из-за смешанного содержимого (Mixed Content). 
Чтобы решить эту проблему:
- **Рекомендуется** настроить Nginx на вашем VPS с бесплатным сертификатом Let's Encrypt (запросы будут идти на `https://yourdomain.com`).
- Или используйте проксирование (Вариант 2), так как в этом случае запросы идут на HTTPS-домен Vercel, а сам Vercel уже обращается к вашему HTTP VPS на стороне сервера.
